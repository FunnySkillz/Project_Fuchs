# App Store Connect Metadata Checklist

Use this checklist before submitting SteuerFuchs to App Store review.

## Required URLs

- [ ] Support URL is set and reachable: `https://steuerfuchs.sebastijan-bogdan.com/support`
- [ ] Privacy Policy URL is set and reachable: `https://steuerfuchs.sebastijan-bogdan.com/privacy`

## App Review Contact

- [ ] Contact name is filled.
- [ ] Contact email is filled.
- [ ] Contact phone is filled.

## App Privacy

- [ ] App Privacy questionnaire completed.
- [ ] "No Data Collected" answers are validated against actual SDK/runtime behavior.
- [ ] OneDrive optional export behavior is accurately described.

## Legal Consistency

- [ ] In-app legal screen text matches website `/privacy` and `/impressum`.
- [ ] Website `/impressum` includes full name, address, email, country, and responsible publisher statement.
- [ ] EN and DE legal text remain semantically equivalent.
